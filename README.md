# Algoritmos e Classificação de Dados - Unipampa Bagé

Repositório dedicado aos trabalhos desenvolvidos na disciplina de **Algoritmos e Classificação de Dados** do curso de **Engenharia de Computação** da **Universidade Federal do Pampa (Unipampa)**, campus **Bagé**.

## Descrição
Este repositório contém os projetos, implementações e materiais relacionados à disciplina de Algoritmos e Classificação de Dados, abordando conceitos fundamentais de estruturas de dados, algoritmos de ordenação, técnicas de busca e análise de complexidade algorítmica.
