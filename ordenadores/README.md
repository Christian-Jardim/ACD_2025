# Programa referente ao Trabalho Prático 1

## Descrição
O foco do programa é a execução de um seleto número de algoritmos de ordenação de vetores e o registro de tempo de execução desses algoritmos.
Os objetos que podem ser armazenados nos vetores são aqueles que implementam a interface Comparable da linguagem Java.
A unidade de tempo de registrio é de milissegundos.
# Os algoritmos implementados são:
- Bubblesort
- Insertionsort
- Selectionsort
- Shellsort
- Heapsort
- Mergesort
- Quicksort
- Countingsort
- Radixsort
- Bucketsort.
