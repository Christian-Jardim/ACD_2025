# Programa referente ao Trabalho Prático 1

## Descrição
O foco do programa é a execução de um seleto número de algoritmos de ordenação de vetores e o registro de tempo de execução desses algoritmos.

# Os algoritmos implementados são:
- Bubblesort
- Insertionsort
- Selectionsort
- Shellsort
- Heapsort
- Mergesort
- Quicksort
- Countingsort
- Radixsort
- Bucketsort.

# Obejetos dos vetrores
Os objetos que poderiam ser armazenados nos vetores deveriam ser aqueles que implementam a interface Comparable do Java
