Programa referente ao Trabalho Prático 2.

O foco do programa é a execução de um seleto número de algoritmos de grafos 
sobre as diferentes representações dos mesmos e o registro de tempo de execução desses algoritmos.

Os algoritmos implementados são: Breadth First Search, Deep First Search, Kruskal Algorithm, Prim Algorithm, Bellman-Ford Algorithm, Dijkstra Algorithm, Floyd-Warshall Algorithm e Ford-Fulkerson.

As representações de grafos implementadas são: Digrafo com Lista de Adjacência e Digrafo com Matriz de Adjacência.

A unidade de tempo de registrio é: milissegundos.
