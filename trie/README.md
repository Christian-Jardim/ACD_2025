# Programa referente ao Trabalho Prático 3
O foco do programa é a simples implementação de um acervo de música utilizando a estrutura de árvore Trie.

# Operações possíveis
As operações possíveis no programa são:
- Busca pelo prefixo de nomes
- Inserção
- Deleção
- Impressão do acervo
- Impressão da árvore.
